## 📌 Описание

Этот проект содержит русификатор интерфейса для **PreSonus Studio One 8 / Fender Studio Pro**.

Русификатор предназначен для русскоязычных пользователей, которым удобнее работать с программой на русском языке.

> ⚠️ Проект не является официальным продуктом PreSonus, Fender или их партнёров.

---

## ✨ Возможности

- ✅ Перевод интерфейса Studio One 8 на русский язык
- ✅ Более понятная навигация по меню
- ✅ Сохранение оригинальной логики программы
- ✅ Подходит для версии **Fender Studio Pro**
- ✅ Простая установка через папку `languages`

## 📦 Установка

### 1. Скачайте русификатор

Скачайте архив из раздела **Releases** или скачайте файлы проекта вручную.

### 2. Откройте папку языков Studio Pro

Обычно она находится здесь:

```text
C:\Program Files\Fender\Studio Pro 8\languages
```

### 3. Скопируйте файлы русификатора

Поместите файл русификатора в папку:

```text
languages
```

### 4. Запустите Studio Pro

После запуска программы выберите русский язык в настройках.

---

## 🔤 Как выбрать русский язык

В верхней панели программы выберите:

```text
Studio Pro → Options → Language
```

Затем выберите русский язык и перезапустите программу, если это потребуется.

---

## 🗑️ Удаление русификатора

Чтобы удалить русификатор:

1. Закройте Studio Pro.
2. Удалите добавленные файлы из папки `languages`.
3. Верните оригинальные файлы из резервной копии, если вы их заменяли.
4. Запустите Studio Pro заново.

---

## 🛡️ Рекомендация перед установкой

Перед заменой файлов рекомендуется сделать резервную копию папки:

```text
C:\Program Files\Fender\Studio Pro 8\languages
```

Например:

```text
languages_backup
```

Это позволит быстро восстановить оригинальные файлы.

## 🧠 Принципы перевода

При переводе используются следующие правила:

| Принцип | Описание |
|---|---|
| Точность | Сохраняется смысл оригинальных пунктов меню |
| Понятность | Термины адаптируются для русскоязычных пользователей |
| Профессиональность | Сохраняется DAW-терминология |
| Минимум вольного перевода | Без лишнего изменения смысла |
| Удобство | Строки подбираются так, чтобы нормально помещаться в интерфейс |

## ❓ Частые вопросы

### Русификатор официальный?

Нет. Это неофициальный пользовательский проект.

### Подойдёт ли он для Studio One 7?

Скорее всего, нет или частично. Основная цель проекта — **Studio One 8 / Fender Studio Pro**.

### Нужно ли заменять оригинальные файлы?

Зависит от способа установки и структуры файлов. Перед любыми изменениями рекомендуется сделать резервную копию.

### Что делать, если после обновления перевод пропал?

Переустановите русификатор или проверьте, не изменилась ли структура папки `languages`.
